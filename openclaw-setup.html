What Is OpenClaw?

OpenClaw is an open-source AI assistant framework that gives you a personal Claude/GPT-powered AI that runs on your own machine or VPS, connects to your Discord server, has access to your files and tools, and can automate tasks on your behalf — 24 hours a day.

💬

Lives in Your Discord

OpenClaw connects directly to your Discord server as a bot. Talk to it just like you'd message a person — it reads and responds in any channel you give it access to.

🧠

Powered by Claude / GPT

Uses the latest Claude or GPT-4 models under the hood. You supply your own API key, so you're in full control of which model and how much you spend.

🔧

Full Tool Access

Can browse the web, read and write files, run shell commands, take screenshots, send messages, execute code — whatever you give it permission to do.

⏰

Runs 24/7 on VPS

On a VPS it stays online permanently. No need to keep your PC on. It can monitor things, post updates, and complete tasks even while you sleep.

🎯

Skills System

Extend it with skills — pre-built packages that give the AI new capabilities like weather, TTS voice, web search, and more. Install them with one command.

🔒

Your Data Stays Private

Because it runs on your own machine, your files and conversations don't go through any third-party servers. Full privacy by default.

Step-by-Step Setup

Once your Windows VPS is running, the full setup takes about 15 minutes. Everything below runs in PowerShell on your VPS — connect via Remote Desktop (RDP) first.

2

Connect to Your VPS via Remote Desktop (RDP)

Your VPS provider will send you the IP address, username, and password. On Windows, open Remote Desktop Connection (search "mstsc" in Start), enter the IP, and log in with your credentials.

Once connected, open PowerShell as Administrator — right-click the Start menu → Windows PowerShell (Admin).

3

Run the OpenClaw Installer

OpenClaw provides a one-line PowerShell installer that handles Node.js detection, installation, and the initial setup wizard automatically.

iwr -useb https://openclaw.ai/install.ps1 | iex

The installer will detect if Node.js 22+ is missing and install it for you. Follow the onboarding prompts when they appear.

💡 Prefer npm directly?

If you already have Node 22+ installed, you can also run:
npm install -g openclaw@latest
openclaw onboard --install-daemon

4

Add Your AI Model API Key

OpenClaw needs an API key to power its AI responses. The easiest option is Anthropic's Claude API. Get a key at console.anthropic.com — Claude Sonnet 4 is the recommended model for a good balance of intelligence and cost.

During onboarding OpenClaw will prompt you for this, or you can configure it manually:

openclaw configure
5

Create a Discord Bot

Go to discord.com/developers/applications, create a New Application, then:

  • Click Bot → Reset Token → copy your bot token
  • Enable Message Content Intent and Server Members Intent
  • Under OAuth2, generate an invite URL with bot + applications.commands scopes and Send Messages, Read Message History, Embed Links, Attach Files permissions
  • Paste the invite URL in your browser to add the bot to your server

Keep your bot token — you'll need it in the next step.

6

Connect OpenClaw to Discord

Back in your VPS terminal, add Discord as a channel. OpenClaw will prompt you for your bot token and guild (server) ID.

openclaw channels add discord

💡 Finding Your Guild ID

In Discord: User Settings → Advanced → enable Developer Mode. Then right-click your server name and click "Copy Server ID".

7

Start the Gateway and Verify

Start OpenClaw's background service (the "gateway") and verify everything is working:

openclaw gateway start
openclaw status
openclaw doctor

If everything is green, head to your Discord server and send your bot a direct message. It should respond within a few seconds.

8

Keep It Running After Reboot

The gateway daemon is installed as a system service by default, so it restarts automatically if your VPS reboots. Verify it's set to auto-start:

openclaw gateway status

Your AI assistant is now live 24/7. Message it in Discord from anywhere and it will respond.

What You Can Do With It

Once OpenClaw is running on your VPS and connected to Discord, the possibilities expand quickly. Here's what our community uses it for:

🎰

Sweepstakes Tracking

Ask it to check for new sites, compare bonus offers, update your spreadsheet, or draft announcements for your community — all from a Discord message.

📢

Discord Announcements

Tell it what you want to announce and it writes it in your style, formats it, and can even post it directly to your Discord channels.

🎬

Video Content

Generate YouTube Shorts and TikTok videos from screen recordings with auto-generated text overlays and captions — ready to upload.

📝

Blog & SEO Articles

Brief it on a topic and it writes full SEO-optimized blog articles matching your site's style, then commits and pushes them to your GitHub repo.

🔔

Heartbeat Monitoring

Set it to check emails, calendar events, and community activity periodically — and proactively ping you in Discord when something needs attention.

🤝

Community Help

In your Discord server it can answer member questions about your guides, link to resources, and handle FAQs while you're busy.
