Step-by-Step Setup

1

Get Your VPS Running

Spin up an Ubuntu 22.04 server. DigitalOcean's $5 droplet handles this easily. SSH in and update everything:

sudo apt update && sudo apt upgrade -y

2

Install Node.js and Dependencies

OpenClaw runs on Node.js. The easiest way:

curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

3

Install OpenClaw

This is where the magic happens:

npm install -g openclaw

That's it. OpenClaw is now installed globally on your system.

4

Create Your Discord Bot

Go to Discord Developer Portal, create a new application, then a bot. Copy the token — you'll need it.

Invite the bot to your server with Administrator permissions (or just the permissions you actually need).

5

Configure and Launch

Run openclaw gateway start and follow the setup wizard. It'll ask for your Discord token, AI API keys, and preferences.

Once configured, OpenClaw starts monitoring Discord and responds to messages. Try sending it a message!
